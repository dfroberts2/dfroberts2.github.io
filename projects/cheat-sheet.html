Ruby

Arrays--Arrays are objects in Ruby that can contain multiple values. They are organized using indices, which are integers that describe the position of a value within the array. The Ruby Docs page here shows the types of methods that can be performed on arrays in Ruby.

Hashes--Hashses are objects in Ruby that also contain multiple values. However, they are organized differently than arrays. Instead of integer indices. They have values associated with keys that cannot be repeated. The Ruby Docs page here shows the types of methods that can be performed on hash objects in Ruby.

Strings--Strings are objects that hold information as a specific order of characters.

Enumerables--The enumerable module provides methods that can operate on collection classes. These are objects that can hold multiple values such as arrays and hashes.

Control Expressions--Control expressions are used to control how a program flows and how the value of different variables changes. The Ruby Docs here demonstrate the different types of controls that can be used in a Ruby program (such as while loops and if statements).

Classes--Classes are first-class objects. When you define a custome made class. You can define methods that apply specifically to that class. The also can have attributes that apply only to that class. The documentation here describes how to create a class.
